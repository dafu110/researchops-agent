# 闭环 Agent 自检

运行以下命令会在临时 JSON 数据目录中完成一次隔离的端到端自检，不会修改项目的常规运行数据：

```powershell
python scripts\run_agent_self_check.py
```

脚本输出结构化 JSON，并逐项验证：

1. 已授权资料范围、引用、`FinalAnswer` 与步骤轨迹。
2. 工具参数、状态、幂等键与调用记录。
3. 高风险请求的审批，以及同一 `run_id` 恢复。
4. 在工具安全边界处的协作式取消。
5. 工具超时与同一运行的失败恢复。
6. 契约 API、指标 API，以及删除后轨迹/审计回放返回 `404` 的历史治理。
7. 32 条评测门禁。

通过结果只证明本地隔离运行的控制闭环。它不替代 PostgreSQL 多实例、真实第三方 MCP、真实写入型连接器、生产凭证、备份恢复和并发故障演练。
